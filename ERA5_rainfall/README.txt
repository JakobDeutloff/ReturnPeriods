This folder contains files downloaded from ERA5 for daily preciptation rate (hourly) in the UK in February 2020.
Data is acquired from https://cds.climate.copernicus.eu/cdsapp#!/software/app-c3s-daily-era5-statistics?tab=overview

The return_period_<date>_<in_years>.nc are the outputs of the script, which indicates whether a certain return period is exceeded on a particular day (above 1: exceeded; below 1: not exceeded)
